This is a [Facebook](https://facebook.com/)-cloned personal project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Stack

- Next.js
- TypeScript
- TailwindCSS
- Prisma (for database)
- Supabase (for backend)
- Motion (prev. Framer Motion)
- Sonner (for toast notifications)
- Zustand (for global state management)
